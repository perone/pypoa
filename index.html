Sobre

A primeira edição do PyPOA objetiva reunir a comunidade Python de Porto Alegre e região e promover a troca de conhecimentos através de uma programação de palestras técnicas para desenvolvedores e interessados na linguagem Python. Participe!

O evento será realizado nas dependências da FTEC, que gentilmente cedeu esse espaço para a comunidade Python de Porto Alegre.

A participação no evento é gratuita e as vagas são limitadas. Além disso, os participantes são incentivados a trazer 2kg de alimentos não perecíveis que serão encaminhandos para uma entidade beneficiente.

A bela imagem que ilustra o site do evento é de autoria de Anderson Vaz.

Time Slot Description

9h30 Abertura -
10h00 Introdução ao Django framework Programador Python at Azion Technologies (http://www.azion.com.br/) Passando por conceitos básicos de um framework até o entendimento básico de modelos de dados, templates e views utilizando Django.
10h50 Python + Celery Em breve.

12h00 Almoço -
14h00 Web stack com Nginx + Chaussette + Circus + Flask Codefi.sh Essa palestra apresenta a configuração de um stack completo para deploy de aplicações WSGI usando Nginx, Chaussette e Circus.
14h20 Machine Learning com Python Codefi.sh Palestra de introdução à Machine Learning usando Python e Scikit-Learn (sklearn).
14h50 Transformando ideias em realidade com web2py CodeFreelas.com Um dos maiores obstáculos para empreendedores não técnicos é transformar a sua ideia em um MVP de qualidade, em pouco tempo e com baixo investimento. Mas esse obstáculo pode ser vencido facilmente com o framework python para desenvolvimento web, o Web2py, que possibilita uma rápida prototipação, e a implementação de ferramentas funcionais seguras e escaláveis com uma curva de desenvolvimento muito pequena.
15h40 Coffee Break -
16h00 Deploy de aplicação Python no OpenShift Gabinete Digital Aprender como utilizar e como fazer deploy de uma aplicação em Python no ambiente open-source OpenShift, criado pela Red Hat. Saiba como criar sua aplicação, como utilizar o git com ela, como guardar as versões do seu deploy, e algumas outras funções.
16h20 Introdução a Automação com Fabric Fabric é uma ferramenta que torna possível administrar e automatizar tarefas em servidores via SSH, neste tutorial será apresentada a biblioteca e alguns exemplos de utilização.
16h50 South, o básico para manter uma "vida saudável" Freelancer Uma breve apresentação sobre a biblioteca South, como utiliza-la em projetos django para manter a evolução constantes dos seus models (apps) de uma forma simples e assim manter uma "vida saudável"
17h30 Encerramento -
